Задача №1459. Римская система счисленияВвод и вывод данных производятся через стандартные потоки ввода-вывода.Дано число X. Требуется перевести это число в римскую систему счисления.Входные данныеДано число X в десятичной системе счисления (1  ≤  X  ≤  100).Выходные данныеВыведите X в римской системе счисления.Примерывходные данные4выходные данныеIVвходные данные19выходные данныеXIXвходные данные46выходные данныеXLVI
