Задача №2959. Знак числаВвод и вывод данных производятся через стандартные потоки ввода-вывода.В математике функция sign(x) (знак числа) определена так:sign(x) = 1,   если x > 0,sign(x) = -1, если x < 0,sign(x) = 0,   если x = 0.Для данного числа x выведите значение sign(x).Входные данныеВводится число x.Выходные данныеВыведите ответ на задачу.Примерывходные данные179выходные данные1
