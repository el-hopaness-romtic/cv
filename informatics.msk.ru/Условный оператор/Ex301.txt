Задача №301. Квадратное уравнениеВвод и вывод данных производятся через стандартные потоки ввода-вывода.Даны действительные числа a, b, c. Найдите все решения квадратного уравнения ax^2 + bx + c = 0.Входные данныеДаны три действительных числа, a не равно 0.Выходные данныеВыведите два действительных числа в любом порядке, если уравнение имеет два корня, одно действительное число – при наличии одного корня. При отсутствии действительных корней ничего выводить не нужно.Примерывходные данные100выходные данные0
